# 🚀 Site Teks Software - Portfólio de Desenvolvimento

**Landing page institucional completa** desenvolvida para a Teks Software (ERP em nuvem).

**🔗 Demo ao vivo**: [tekssoftware.com.br](https://tekssoftware.com.br)

---

### ✅ O que foi desenvolvido
- Landing page moderna e 100% responsiva (mobile-first)
- Blog integrado com artigos sobre ERP e gestão
- Páginas de produtos (Teks Food, Varejo, Posto, Cardápio Digital)
- SEO completo (meta tags, sitemap, dados estruturados)
- Integração WhatsApp Business
- Animações suaves com CSS puro + JavaScript vanilla
- Performance otimizada (imagens compactas e lazy loading)
