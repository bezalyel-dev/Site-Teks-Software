# Site Teks Software

Site institucional da Teks Software - Sistema ERP em Nuvem para Varejo, Food Service e Postos de Combustível.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Estrutura

- `index.html` - Página principal
- `blog/` - Artigos do blog
- `siteimgs/` - Imagens do site
- `clientes/` - Imagens de clientes
- `depoimentos/` - Vídeos e imagens de depoimentos

## Deploy

Este site está hospedado no GitHub Pages.
